# Images

Place image files here and reference them from the data in `index.html`.

Expected filenames for the French Quarter sample dataset:

| File | Place |
|---|---|
| `jackson-square.jpg` | Jackson Square |
| `st-louis-cathedral.jpg` | St. Louis Cathedral |
| `cafe-du-monde.jpg` | Café du Monde |
| `french-market.jpg` | French Market |
| `royal-street.jpg` | Royal Street |
| `bourbon-street.jpg` | Bourbon Street |
| `preservation-hall.jpg` | Preservation Hall |
| `thnoc.jpg` | Historic New Orleans Collection |
| `jean-lafitte-visitor-center.jpg` | Jean Lafitte NHP Visitor Center |
| `moon-walk.jpg` | Moon Walk & Mississippi Riverfront |

If an image file is missing, the page will display a placeholder gracefully.

## Recommended sizes

- Card thumbnails are displayed at 16:9 aspect ratio
- Images inside the accordion gallery are 4:3
- Aim for 800–1200px wide, optimized for web (JPEG or WebP)
